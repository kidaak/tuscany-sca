Installation Instructions

Here you can find everything you need to know to install Apache Tuscany.

If you haven't already, the first step would be to download the latest release of Apache Tuscany from our Downloads page.

Configuring Apache Tuscany

You can find detailed instructions on how to configure your Tomcat environment, or a Stand-Alone environment from the Samples README. Please note that if you want to run Tuscany on your Tomcat environment, you will need to use Tomcat 5.5 or later because those versions support JDK 5.0.
